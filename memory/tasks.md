  1. `PolygonEditor.tsx` : rappeler `draw.changeMode('draw_polygon')` de façon synchrone depuis le gestionnaire de l'événement `draw.create` lui-même fait re-rentrer Mapbox GL Draw dans son propre cycle interne (`onStop`/`fire`) → `RangeError: Maximum call stack size exceeded`.   2.   3. `contractCreation.schema.ts` : `accumulationMaximaleCm` (champ "Accumulation maximale" de l'étape Obligations) passait silencieusement de `null` à `0` après un aller-retour entre étapes (remontage du champ non-contrôlé par React Hook Form) — `0` échouait la contrainte Zod `.positive()`, ce qui faisait échouer `handleSubmit` **silencieusement** (aucun toast, aucune erreur console, le bouton "Créer" semblait ne rien faire).
